# Code Katas

## Requirements

  - MacOS
  - Docker 17.09+

## Setup

  - git clone this repository
  - run `script/bootstrap`
  - open project in PhpStorm
  - run "Unit tests" from Run configurations

## Usage

### Docker container console

    script/console
